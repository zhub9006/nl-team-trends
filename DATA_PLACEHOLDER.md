# NL Team Trends — Data File Index

## Available Data Files

| File | Description | Key Content |
|------|-------------|-------------|
| `data/nl_historical_performance.csv` | Season-by-season NL standings | NL champion, 2nd place, WS outcomes 1876-2025 |
| `data/nl_all_time_records.csv` | Franchise-level totals across all NL seasons | 15 NL franchises with games, wins, losses, Win%, pennants, WS titles |
| `data/nl_championship_trends.csv` | Championship distribution by era | 7 historical eras with dominant teams, pennant counts, WS outcomes |
| `data/nl_notable_records.csv` | Single-season & franchise milestones | Best records, longest streaks, era highlights |

## Data Conventions

- **Win%** = Wins / Games Played
- **Pre-1903 pennants** = NL championship only (no World Series before 1903)
- **154G era** = 1892-1961 (154 games per season)
- **162G era** = 1962-2019, 2021-present
- **COVID season** = 2020 (60 games)
- **Split seasons** = 1892 & 1981 (listed as separate entries)
- **Franchise totals** include pre-relocation data (SF Giants = NY Giants + SSF Giants)
- **Brewers** counted as NL from 1998 (moved from AL Central)
- **Expos** = 1969-2004 season data; combined with Nationals from 2005

## Missing Data Notes

- Pre-1892 W-L data has gaps in some seasons (varied schedule lengths)
- Individual season batting/pitching stats: use SABR Lahman Database CSV files
- Detailed player-level data: see SABR Lahman Database (sabr.org)
