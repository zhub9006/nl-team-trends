# Data Files Index - nl-team-trends

Data files in data/:
1. nl_historical_performance.csv - Year-by-year NL champion data 1876-2025
2. nl_all_time_records.csv - All-time franchise records with W-L totals
3. nl_notable_records.csv - Key records with context
4. nl_championship_trends.csv - Era-based championship trends
5. nl_notable_seasons.csv - Significant seasons with key stats
