# Source References for NL Team Trends Research

This document provides detailed attribution for all data sources used in this repository.

---

## Primary Sources

### 1. Baseball-Reference — NL Team Wins Records
- **URL**: https://www.baseball-reference.com/leagues/NL/index.shtml
- **Coverage**: 1876–present
- **Data Used**: Year-by-year NL standings, team stats, game-by-game records
- **Authority**: Gold standard for MLB statistics; maintained by Sports Reference

### 2. Baseball Almanac — NL Team vs Team Win-Loss (1876–2026)
- **URL**: https://www.baseball-almanac.com/teams/teamvsteam-nl.shtml
- **Coverage**: 1876–2026
- **Data Used**: 15×15 head-to-head W-L matrix for every NL team vs every other NL team
- **Authority**: Comprehensive historical baseball research site established 1999

### 3. Baseball Almanac — Year-by-Year History
- **URL**: https://www.baseball-almanac.com/yearmenu.shtml
- **Coverage**: 1876–present
- **Data Used**: NL leadership, W-L records, fabulous feats per season

### 4. SABR Lahman Baseball Database
- **URL**: https://sabr.org/lahman-database/
- **Coverage**: 1871–2025 (including Negro Leagues)
- **Data Used**: Complete batting/pitching/standings/team stats/managerial/postseason data
- **Authority**: Created by SABR member Sean Lahman

### 5. Baseball Data Hub — All MLB Seasons
- **URL**: https://baseballdatahub.com/seasons/
- **Coverage**: 1871–2026
- **Data Used**: Complete season standings, statistical leaders, postseason results
