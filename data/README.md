# NL Team Trends - Data Files Directory

This directory contains compiled historical National League performance data.

## File Descriptions

| File | Description |
|------|-------------|
| `nl_all_time_records.csv` | All-time franchise win-loss totals for all 15 NL teams |
| `nl_championship_trends.csv` | Championship highlights organized by era (8 eras) |
| `nl_notable_records.csv` | Single-season and franchise records with context |
| `nl_recent_standings.csv` | Full division champion standings 2020–2025 |
| `source_references.md` | Detailed attribution of all data sources |

## How to Use
- Load CSVs directly with `pandas.read_csv()`
- Use `source_references.md` for citation in any analysis
- See `docs/data_notes.md` for methodology and conventions
- See `visualizations/README.md` for visualization roadmap and starter code
