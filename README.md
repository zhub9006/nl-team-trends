# NL Team Trends

Comprehensive research project compiling historical **National League (NL)** team performance data, win-loss records, season trends, and championship history from 1876 to present.

The National League is baseball's oldest professional league, founded in 1876. This repository includes all-time franchise records, pennant winners, and team-vs-team matrices.

## Data Sources

- Baseball Almanac: NL team-vs-team win-loss (1876-2026)
- Baseball Reference: Year-by-year NL standings
- StatMuse: All-time aggregate team statistics
- Grokipedia: Franchise win-loss records through 2025
- Baseball Data Hub: All MLB seasons 1871-2026
