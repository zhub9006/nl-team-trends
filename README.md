# NL Team Trends Repository - Key Updates

## What's New

1. **Comprehensive NL Pennant Winners (1876-2025)**: Year-by-year NL champion with wins, losses, WIN%, 2nd place, and World Series results. Includes era classifications.

2. **Championship Trends by Decade/Era**: Aggregated view of which teams dominated each era with pennant counts, WS titles, and average winning percentages.

3. **Recent NL Standings (2020-2025)**: Complete divisional breakdown for all 3 NL divisions including Wild Card data and WS outcomes.

4. **All-Time Franchise Detailed Records**: Granular franchise statistics with era trends and key insights.

## Data Notes
- Pre-1961: 154-game schedule
- 1962-2019: 162-game schedule
- 2020: 60-game COVID season
- All data through 2025 season end
- Some 1940s/1950s WS MVP data may need verification against Baseball Reference
