# NL Team Trends

Comprehensive research project compiling historical **National League (NL)** team performance data, win-loss records, season trends, and championship history from 1876 to present.

The National League is baseball's oldest professional league, founded in 1876. This repository includes all-time franchise records, pennant winners, team-vs-team matrices, and season-by-season data for analyzing team dominance, franchise trajectories, divisional realignment impacts, and the evolving NL competitive landscape.

## Data Sources

| Source | Coverage | Description |
|--------|----------|-------------|
| Baseball Almanac | 1876-2026 | NL team-vs-team win-loss matrices |
| Baseball Reference | 1876-present | Year-by-year NL standings & leaders |
| Baseball Data Hub | 1871-present | Complete season standings & stats |
| Lahman Database (SABR) | 1871-2025 | Batting, pitching, standings & team stats |
| Grokipedia / Wikipedia | 1876-2025 | All-time franchise win-loss records |
