# NL Team Trends - Enhanced Documentation

This repository now contains comprehensive historical National League performance data, including season-by-season records, decade-level statistics, and recent standings from 1876 through 2025.

## Updated Feature: Season-by-Season NL Performance Data

The new `nl_season_by_season_records.csv` file captures:
- Yearly NL champion performance (W-L, Win%, Runs, Division)
- Era classification for each season
- Runs scored differential and percentage rank
- Division placement within the NL structure

## Research Sources & Data Pipeline

Primary sources: [Baseball Reference NL](https://www.baseball-reference.com/leagues/NL/), [SABR Lahman DB](https://sabr.org/lahman-database/), [Baseball Data Hub](https://baseballdatahub.com/seasons/), [Linger & Look](https://www.lingerandlook.com/Names/BaseballStandings.php)

## Quick Stats Summary

- **Oldest franchise**: Chicago Cubs (1876-present)
- **Most pennants**: LA Dodgers (26 through 2025)
- **Most WS titles (NL)**: St. Louis Cardinals (11)
- **Dodgers dynasty**: 8 straight NL West titles (2018-2025)
- **Best 162-game record**: 2022 LA Dodgers (111-51)
- **Biggest playoff upset**: 1914 Boston Braves (Miracle Braves)
