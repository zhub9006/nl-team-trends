# NL Team Trends

Comprehensive research project compiling historical National League (NL) team performance data, win-loss records, season trends, and championship history from 1876 to present — for analyzing team dominance, franchise trajectories, and the evolving NL competitive landscape.

## Quick Start

```bash
git clone https://github.com/zhub9006/nl-team-trends.git
cd nl-team-trends
pip install -r requirements.txt
```

## Data Files

| File | Description |
|------|-------------|
| `data/nl_all_time_records.csv` | All-time franchise records (W/L/G/Win%/WS titles) |
| `data/nl_historical_performance.csv` | Year-by-year NL standings 1876-2025 |
| `data/nl_pennant_winners.csv` | NL pennant winners 1876-2025 |
| `data/nl_championship_trends.csv` | Championship highlights by era |
| `data/nl_notable_records.csv` | Key records by category |
| `data/nl_h2h_rivalries_detailed.csv` | Head-to-head rivalry data with era context |

## Key Findings

- **Most WS titles**: St. Louis Cardinals (11)
- **Most NL pennants**: LA Dodgers (26)
- **Most all-time NL wins**: SF Giants (11,663)
- **Longest dynasty**: Atlanta Braves (14 straight div titles, 1991-2005)
- **Most lopsided H2H**: Cardinals vs Pirates (60.6% Cardinals)
- **Most one-sided modern**: Dodgers vs Cubs (69.5% Dodgers)
