## Usage

```python
import pandas as pd

# Load NL performance data
df = pd.read_csv('data/nl_historical_performance.csv')

# Aggregate by era
era_stats = df.groupby(['era', 'year'])['win_pct'].mean()

# Compare teams over time
team_stats = df.groupby('team').agg({'wins': 'sum', 'losses': 'sum'})
team_stats['win_pct'] = team_stats['wins'] / (team_stats['wins'] + team_stats['losses'])
```
