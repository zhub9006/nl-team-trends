# Data Notes — nl-team-trends

## Methodology
- **Win%** = Wins / (Wins + Losses), ties excluded post-1920
- Franchise totals include pre-relocation segments
- Milwaukee Brewers: AL→NL in 1998
- Washington Nationals: Expos (1969-2004) + DC (2005+)
- LA Dodgers: Brooklyn (1884-1957) + LA (1958+)
- SF Giants: NY Giants (1883-1957) + SF Giants (1958+)
- Atlanta Braves: Boston Red Stockings (1871-1952) + Milwaukee (1953-1965) + Atlanta (1966+)

## Schedule Eras
- 1876-1884: 60-112 games
- 1885-1908: 126-154 games
- 1909-1961: 154 games
- 1962-2019: 162 games
- 2020: 60 games (COVID)
- 2021+: 162 games

## Conventions
- Pre-1903: No WS; pennant winners only
- 1994: Strike voided; no championship
- Split seasons: 1892 and 1981 handled with half-designators
