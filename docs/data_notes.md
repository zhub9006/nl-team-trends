# Data Notes

Methodology, conventions, and limitations for the NL Team Trends project data.

## Methodology
- Primary data sources: Baseball-Reference, Baseball Almanac, SABR Lahman Database
- Secondary sources: StatMuse, Wikipedia, Grokipedia, AP News
- All data verified against at least two independent sources

## Conventions
- Team names: Canonical NL names; franchise movements noted parenthetically
- W-L format: Wins-Losses (e.g., 98-55); no ties in modern data
- Win%: W/(W+L), 3 decimal places; ties excluded
- Schedule: 154G (1909-1961) → 162G (1962-2019) → 60G (2020) → 162G (2021+)
- Franchise continuity: Relocated teams treated as continuous entities

## Limitations
1. Pre-1900 data may have gaps in statistical categories
2. 2026 data is incomplete (season in progress as of July 2026)
3. Win percentages not adjusted for era-specific factors
4. Interleague play data excluded from franchise-vs-franchise comparisons
