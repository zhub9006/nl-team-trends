# NL Team Trends — Data Notes

## Methodology

### Data Collection
- Primary data sources: Baseball-Reference, Baseball Almanac, SABR Lahman Database
- Secondary sources: StatMuse, Wikipedia, Grokipedia, AP News
- All data was verified against at least two independent sources

### Conventions
#### Team Names
- Use canonical NL team names; franchise movement history noted in parentheses
- Examples: "Brooklyn Superbas" (1899-1931) → "Brooklyn Dodgers" (1932-1957) → "Los Angeles Dodgers" (1958-present)
- "Boston Beaneaters" → "Boston Doves" → "Boston Rustlers" → "Boston Braves" → "Boston Bees" → "Milwaukee Braves" → "Atlanta Braves"
- "Chicago White Stockings" (1876-1890) → "Chicago Colts" → "Chicago Orphans" → "Chicago Cubs" (1902-present)
- "Cincinnati Red Stockings" (1876-1880) / "Cincinnati Reds" (1890-present)

#### Winning Percentage
- Calculated as W / (W + L), rounded to 3 decimal places
- Ties excluded from the denominator

#### Schedule Lengths
- 1876-1884: 60-112 games; 1885-1908: 126-154; 1909-1961: 154; 1962-2019: 162
- 2020: 60 games (COVID-19); 2021-2025: 162

#### 1994 Strike
- The 1994 season was cancelled; no pennant or WS was awarded

#### Franchise Continuity
- Relocated teams treated as continuous entities (e.g., NY Giants + SF Giants)

---

## Limitations
1. Pre-1900 data may have gaps in statistical categories
2. 2026 data is incomplete (season in progress)
3. Win percentages not adjusted for era-specific factors
4. Interleague play data excluded from franchise-vs-franchise comparisons
